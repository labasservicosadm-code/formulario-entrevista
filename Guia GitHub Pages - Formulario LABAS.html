LABAS — Formulário Operacional

Publicar o formulário na internet
e acessar por link fixo

Usando o GitHub Pages — gratuito, sem prazo de expiração, abre em qualquer celular ou computador com um link salvo nos favoritos.

⏱ ~10 minutos 💰 100% gratuito 🔗 Link fixo permanente 📱 Funciona offline depois do primeiro acesso
💡
O que é o GitHub Pages? É um serviço gratuito do GitHub que transforma um arquivo HTML em uma página web com link fixo. Não precisa entender de programação — você só vai criar uma conta, fazer o upload do arquivo e ativar a publicação.
FASE 1
Criar conta no GitHub
1
Acesse o GitHub — github.com

Abra o navegador e acesse github.com. Clique no botão Sign up no canto superior direito.

🔒 github.com
GitHub
Where the world builds software
Sign in Sign up
2
Preencha o cadastro

O GitHub vai pedir algumas informações básicas:

  • E-mail: use seu Gmail normalmente
  • Senha: crie uma senha forte (mínimo 8 caracteres, com letra e número)
  • Username (nome de usuário): esse nome vai aparecer no link do seu formulário. Use algo simples como labas-rh ou labasservicos — sem espaços, sem acento, tudo minúsculo.
  • Complete o captcha (verificação de segurança) e clique em Continue
  • Confirme o e-mail — o GitHub vai mandar um código no seu Gmail
✅
O username que você escolher vai fazer parte do link final. Ex: se escolher labas-rh, o link será labas-rh.github.io/formulario-entrevista
FASE 2
Criar o repositório
📁
O que é um repositório? É simplesmente uma pasta no GitHub onde você vai guardar o arquivo do formulário. Pense como uma pasta no Google Drive, só que na internet.
3
Criar novo repositório

Depois de entrar na conta, clique no ícone "+" no canto superior direito → New repository

FASE 3
Fazer upload do formulário
4
Renomear o arquivo HTML antes do upload

Antes de fazer o upload, renomeie o arquivo para index.html. Isso é importante — o GitHub Pages abre automaticamente o arquivo chamado index.html.

  • Localize o arquivo Formulario Entrevista LABAS.html na pasta Downloads
  • Clique com o botão direito → Renomear
  • Mude o nome para index.html e confirme
5
Fazer o upload do arquivo

Dentro do repositório criado, clique em Add file → Upload files

🔒 github.com/labas-rh/formulario-entrevista
📄
Arraste o arquivo index.html aqui
ou clique para selecionar
Commit changes
Add formulario entrevista
Commit changes
  • Arraste o arquivo index.html para a área pontilhada ou clique para selecionar
  • Em "Commit changes", deixe o texto padrão ou escreva Add formulario entrevista
  • Clique em Commit changes
FASE 4
Ativar o GitHub Pages
6
Abrir as configurações do repositório

Dentro do repositório, clique em ⚙️ Settings na barra de abas superior.

No menu da esquerda, role até encontrar 📄 Pages e clique.

7
Ativar o Pages
🔒 github.com/labas-rh/formulario-entrevista/settings/pages
GitHub Pages
Source
Branch: main ▾
/ (root) ▾
Save
✅ Your site is live at https://labas-rh.github.io/formulario-entrevista/
  • Em Source, selecione o branch main (já deve estar selecionado)
  • Na pasta, deixe / (root)
  • Clique em Save
  • Aguarde 1 a 2 minutos — recarregue a página e vai aparecer o link verde com o endereço do seu formulário
✅
Quando aparecer a mensagem verde "Your site is live at...", está publicado. Copie o link — é o link fixo do formulário.
FASE 5
Salvar nos favoritos e usar
8
Salvar o link nos favoritos

Com o link do formulário aberto no navegador:

💻 Computador (Chrome) Pressione Ctrl + D → nomeie como Entrevista LABAS → Salvar na barra de favoritos
📱 Celular Android (Chrome) Toque nos três pontinhos (⋮) → Adicionar à tela inicial → vira ícone como app
🍎 iPhone (Safari) Toque em compartilhar → Adicionar à Tela de Início → vira ícone como app
💬 WhatsApp (recomendado) Mande o link para si mesmo em uma conversa fixada — acesso rápido em qualquer situação
9
Como atualizar o formulário no futuro

Se eu gerar uma versão nova do formulário, você só precisa fazer o upload do novo arquivo — o link permanece o mesmo.

  • Renomeie o novo arquivo para index.html
  • Entre no GitHub → repositório formulario-entrevista
  • Clique no arquivo index.html existente → ícone de lápis ✏️ → ou arraste o novo arquivo por cima
  • Clique em Commit changes
  • Aguarde 1 minuto e recarregue — o link já mostra a versão nova
